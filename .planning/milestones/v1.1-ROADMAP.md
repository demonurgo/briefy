## Milestone Summary

**Key Decisions:**
- Anthropic PHP SDK direct (no Prism abstraction) — simpler dependency, direct streaming
- SSE via Laravel streaming response + React EventSource/fetch — no WebSocket for AI streams
- BYOK per-org key resolution via AnthropicClientFactory — testable with AnthropicClientInterface
- organization_user pivot (replacing users.organization_id scalar) — enables multi-org membership
- recharts ^3.8.1 chosen for charts — lightweight, tree-shakable, no canvas dependency
- ActivityLog timestamps=false + user_id nullable+nullOnDelete — lean schema, safe on user delete

**Issues Resolved:**
- Phase 03: DashboardPlanningWidget.tsx missing from initial verification — created and verified
- Phase 04: intervention/image package missing from vendor despite being in composer.json — installed in 2026-04-24 UAT

**Issues Deferred:**
- RT-01–RT-05: Real-time collaboration (live comments, notifications) — deferred to v1.2
- SSE consolidation (useAiStream + EventSource) — deferred to v1.2 (WARNING 8)
- AI conversation picker dropdown — deferred to v1.2
- Multi-org creation UI in /settings — deferred to v1.2 (found during UAT)

**Technical Debt Incurred:**
- TypeScript strict errors (auth.organization shape mismatch) — pre-existing, non-blocking
- AiIcon size enum doesn't include 11/14 — minor, several call sites use non-enumerated sizes
