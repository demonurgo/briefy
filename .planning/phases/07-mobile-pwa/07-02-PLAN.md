| Boundary | Description |
|----------|-------------|
| touch event → triggerClose | onTouchMove reads clientY from touch event. This is client-only gesture handling with no network calls or state mutations beyond calling onClose. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-02 | Denial of Service | onTouchMove handler | accept | The handler only reads touch coordinates and calls triggerClose() — no network requests, no loops, O(1) cost per event. Cannot be weaponized by an attacker. |
| T-07-03 | Spoofing | triggerClose setTimeout | accept | setTimeout(onClose, 200) is a 200ms UI delay for animation. No security surface — onClose only updates React state in the parent. |
